Steps to run the model:
1. Open the code in colab environment.
2. Upload the fine_tuned_new_model.zip in the colab.
3. Run the code until section: "Meeting Summarizer." Skip the section for fine tuning.
4. Run the " Loading the fine tuned model" to load the fine tuned model uploaded in step3.
5. Run the section :" Fine Tuned Meeting Summarizer". 
6. Once step is completed successfully it will generate an url. 
7. Click on this generated url and it will open the UI.
